How to install and use

Follow these steps:

  1. Install Witchcraft Chrome extension: get it here
  2. Install Web Server Chrome app: get it here
  3. Choose a folder where your scripts will live. For instance, ~/witchcraft-scripts (create it now if it doesn't exist yet)
  4. Open Web Server Chrome app configuration window. Go to the chrome://apps/ window and open it. It looks like this:
  5. Configure Web Server Chrome app
    1. Click the CHOOSE FOLDER button and select the folder you chose in step 3
    2. Toggle on the server if it isn't yet (should say Web Server: STARTED)
    3. Mark Run in background
    4. Mark Start on login
    5. On Enter Port, write 5743
  6. Restart Chrome for the changes to apply and Witchcraft should be working now. Head to http://localhost:5743/ and you should see your list of scripts (or an empty folder if you haven't added any scripts yet!)

To test it, create some script, put it in the selected folder and try it!

Note: if the instructions didn't work for any reason, you can get help by posting you problem here.
